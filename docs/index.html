Cognitive Obesity

Cognitive-Experiential Imbalance as a Unifying Framework for Depression, Violence, and Information Overload

Author: Kazuyoshi Miyauchi Year: 2026 Status: Preprint · PsyArXiv Data: 177 countries × 1990–2023
Key Findings ↓ Replication Repository Preprint EN (PDF) Preprint JA (PDF)

The Core Idea

The mental health impact of digital technology depends not on how much time you spend on screens, but on the balance between involuntary cognitive input and experiential processing — classified by the degree of sensorimotor loop closure with the environment.

L = α₁ · I − α₂ · C
TERM 1 — α₁·I
Cognitive Input
Low loop closure: scrolling, push notifications, algorithmic feeds, targeted ads
TERM 2 — α₂·C
Experiential Processing
High loop closure: exercise, creative work, face-to-face interaction, embodied play

Key Findings

Block B · Macro

Ad proxy survives first-difference

Internet×GDP/capita retains predictive power for depression after controlling for GDP change. Δproxy t=3.19 ΔGDP t=0.16 n.s.

Block B · Macro

Suicide shows threshold, depression doesn't

Hansen PTR detects a significant protective saturation point for suicide but not for depression. Suicide γ*=25.7, F=213 Depression p=0.654

Block C · Individual

Additive model wins decisively

L = α₁·I − α₂·C validated: additive model beats ratio R = I/C model. ΔAIC = 86 N = 21,736

Block C · Individual

No processing = worst outcomes

High passive input with no exercise or active leisure produces 2× the Fair/Poor health rate. 25.2% vs 12.5% p=8.57×10⁻¹⁶

Limitations (Honest Assessment)

This is exploratory research. The following limitations constrain all findings:

△ All individual data cross-sectional — causal direction unidentified
△ Ecological fallacy constrains all macro findings
△ Ad proxy construct validity limited (indirect measure)
△ Depression reversal point in upper ~10th percentile only
△ No natural experiment tied to advertising ecosystems
△ US-only individual data — cross-cultural generalizability unknown
