# 因果言語監査報告書 (Causal Language Audit)

**対象論文:**
- 日本語版 v9_4 (152KB, 38ページ)
- 英語 preprint (30KB, 18ページ)

**監査日:** 2026-03-06
**監査基準:** 観察データから因果的主張への飛躍、hedging の不足、ecological fallacy のリスク

---

## 分類基準

| コード | 定義 |
|--------|------|
| **CAUSAL_OVERCLAIM** | 観察・相関データから因果的結論を導いている |
| **HEDGING_NEEDED** | 主張自体は合理的だが、不確実性の表現が不十分 |
| **CAUSAL_APPROPRIATE** | データの性質に見合った適切な因果言語 |
| **DIRECTIONAL_LANGUAGE** | 方向性を暗示する言語（「予測する」「駆動する」等）が因果推論と混同されうる |

**深刻度:**
- **Critical**: 論文の中核的主張に関わる因果的過剰主張
- **Major**: 重要な分析結果の解釈における因果言語の問題
- **Minor**: 表現上の問題だが、文脈から著者の意図は理解可能

---

## Critical (深刻)

### C-01: Abstract -- 「規定因」の使用
- **所在:** 日本語版 p.2 (Abstract)
- **原文:** 「精神病理の**規定因**はスクリーンタイムの絶対量ではなく、非選択的認知入力と体験的処理のバランス（加法的不均衡）であり」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 「規定因」(determinant) は因果的決定要因を意味する。本論文のエビデンスはマクロパネル相関と横断的個人データであり、因果的決定要因を同定するに足るデザインではない。Abstract は論文全体の印象を決定するため、ここでの overclaim は特に深刻。
- **修正案:** 「精神病理の**関連要因として最も重要なのは**スクリーンタイムの絶対量ではなく...である**可能性を提示する**」

### C-02: Abstract -- 「the determinant of psychopathology」
- **所在:** 英語版 p.1 (Abstract)
- **原文:** "The core claim is that **the determinant** of psychopathology is not the absolute amount of screen time but the balance..."
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** C-01の英語版。"determinant" は因果的決定因を明示的に主張している。
- **修正案:** "The core claim is that **a key correlate** of psychopathology..." or "the **proposed primary factor** underlying psychopathology..."

### C-03: Abstract -- 「確認された」(confirmed)
- **所在:** 日本語版 p.2
- **原文:** 「診断バイアスが支配的であることが、ハードアウトカム（自殺率）との比較で**確認された**」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** マクロパネルデータの比較で「確認された」は過剰。alternative explanations が排除されていない。
- **修正案:** 「...との比較で**示唆された**」

### C-04: Abstract -- 「demonstrates that the proxy captures information beyond GDP」
- **所在:** 英語版 p.1
- **原文:** "...surviving first-difference analysis controlling for GDP change....**demonstrating** that the proxy captures information beyond GDP"
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 1階差分分析で生存したことは "demonstrates" ではなく "suggests" or "is consistent with" の水準。マクロデータの1階差分は多くの交絡を除去しない。
- **修正案:** "...**suggesting** that the proxy captures information beyond GDP"

### C-05: Abstract -- 「confirms that physical exercise and active cognitive leisure independently predict wellbeing」
- **所在:** 英語版 p.1
- **原文:** "Individual-level validation (NHANES N = 5,032; ATUS N = 21,736) **confirms** that physical exercise and active cognitive leisure **independently predict** wellbeing"
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** NHANES と ATUS はいずれも横断的観察データ。"confirms" は事前仮説の因果的検証を含意する。"independently predict" は回帰での統計的有意性を指すが、横断データでは因果的独立性を意味しない。逆因果（健康な人ほど運動する）が排除されていない。
- **修正案:** "Individual-level analysis....**shows cross-sectional associations** between physical exercise, active cognitive leisure, and wellbeing as **additive correlates**"

### C-06: 知見1の解釈 -- 「駆動される」
- **所在:** 日本語版 p.5 (Section 2.1.2)
- **原文:** 「低所得国では鬱病と殺人がそれぞれ独立の外的原因（貧困、感染症、政治的不安定等）により**駆動される**ため」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 「駆動される」は明確な因果言語。国レベルの相関パターンから個別アウトカムの因果的駆動因を特定することは ecological fallacy。
- **修正案:** 「低所得国では鬱病と殺人がそれぞれ独立の外的要因...と**関連している可能性があり**」

### C-07: 知見1の解釈 -- 「処理容量という潜在的共通因子が両アウトカムを支配する」
- **所在:** 日本語版 p.5
- **原文:** 「高所得国ではこれらの外的駆動因が抑制され、個人の処理容量という潜在的共通因子が両アウトカムを**支配する**ようになる」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 国レベルの相関構造の変化から「支配する」という因果的結論を導いている。「処理容量」は直接測定されておらず、推測された潜在変数。
- **修正案:** 「...個人の処理容量という潜在的共通因子が両アウトカムの**主要な関連要因となる可能性がある**」

### C-08: 2.2.1 -- ラグ分析の因果的解釈
- **所在:** 日本語版 p.10-11 (Section 2.2.1)
- **原文:** 「順方向の因果（第一近似R → 鬱病）は統計的に支持される」「逆方向の因果（鬱病 → 第一近似R）は符号が逆であり**棄却される**」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** Granger因果性検定は「予測的先行関係」(predictive precedence) であり、真の因果関係の証明ではない。著者自身が「本分析は厳密な因果推論ではなく予測的先行関係の検証である」と後述しているが、上記の主文では明確に「因果」と書いている。さらに、マクロパネルデータのGranger検定は、国レベルの時変交絡（政策変更、技術進歩の共通トレンド等）を排除できない。
- **修正案:** 「順方向の**予測的先行関係**（第一近似R → 鬱病）は統計的に支持される」「逆方向の予測的先行関係は符号が逆であり、**単純な逆因果仮説とは整合しない**」

### C-09: 1階差分分析 -- 「GDPから独立した予測力を持つことが確認された」
- **所在:** 日本語版 p.14-15 (Section 2.2.5.2)
- **原文:** 「広告曝露プロキシはGDP の代理変数ではなく、GDP成長に含まれない情報...を捕えている」
- **問題類型:** HEDGING_NEEDED
- **問題:** 1階差分でGDPと分離できたことは重要だが、「捕えている」は断定的すぎる。proxy = Internet x GDP/capita であるため、GDP自体との共線性の問題は構造的に内在している。
- **修正案:** 「...を捕えている**可能性がある**」

### C-10: Section 6.4 -- RCTの再解釈における因果的飛躍
- **所在:** 日本語版 p.31 (Section 6.4), 英語版 p.13-14
- **原文:** 「RCT でランダム化されている以上、この第2 項の増大が鬱症状の改善を**因果的に引き起こしている**ことになる」/ "Since exercise is randomized, this Term 2 increase **causally produces** depression improvement."
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 運動RCTは「運動 → 鬱改善」の因果関係を示すが、「運動 → Term 2の増大 → 鬱改善」という**本枠組みのメカニズム**を通じた因果連鎖を示すものではない。運動の効果はBDNF、炎症マーカー、社会的接触、自己効力感など多数のメカニズムで説明可能。他者のRCTデータを自説の特定メカニズム（「ループ閉合度」「体験的処理」）の因果証拠として再解釈するのは論理的飛躍。著者は「排他的ではなく統合的に理解できる」と注記しているが、主文の因果的断言が強すぎる。
- **修正案:** 「運動がランダム化されている以上、運動という介入が鬱症状を因果的に改善することは示されている。本枠組みでは、この効果を第2項（体験的処理）の増大として**解釈する**が、この**特定のメカニズム経路**は本データからは**直接検証されていない**」

### C-11: Section 2.3 Block C -- 横断データからの因果的結論
- **所在:** 日本語版 p.18 (Section 2.3.3)
- **原文:** 「Fair/Poor 健康率の2 倍の差は、認知入力の過剰と体験的処理の不足が複合する状態...が最も健康転帰が**悪いことを示す**」
- **問題類型:** HEDGING_NEEDED
- **問題:** ATUS は1日調査の横断データ。「示す」は因果的含意を持ちうる。健康状態が悪い人は運動できず受動的余暇が増えるという逆因果が排除されていない。
- **修正案:** 「...が最も健康転帰が悪いことと**関連していることを示す（ただし横断データのため因果方向は未確定）**」

## Major (重大)

### M-01: 導入 -- 「認知の肥満は認知入力と体験的処理のバランス不均衡から生じる」
- **所在:** 日本語版 p.4 (Section 1)
- **原文:** 「認知の肥満は認知入力と体験的処理のバランス不均衡から**生じる**」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 「生じる」は因果的生成を意味する。この段階で枠組みは仮説であり、因果的生成は未証明。
- **修正案:** 「認知の肥満は認知入力と体験的処理のバランス不均衡として**定義される**」（定義として提示する）

### M-02: 知見2の解釈 -- 「整合する」の過剰使用
- **所在:** 日本語版 p.6 (Section 2.1.2)
- **原文:** 「この結果は本枠組みの中核的主張...と**整合する**」
- **問題類型:** HEDGING_NEEDED
- **問題:** 「整合する」(consistent with) 自体は適切な hedging だが、論文全体を通じて「整合する」が繰り返し使用され（20回以上）、累積的に因果的印象を与えている。個々には適切でも、全体として confirmation bias 的な印象を与える。
- **修正案:** 一部を「**矛盾しない**」「**一つの可能な解釈である**」「**説明可能だが唯一の説明ではない**」等に多様化する

### M-03: 知見3の解釈 -- 「共通の基盤から生じる異なる発現形である」
- **所在:** 日本語版 p.7 (Section 2.1.2)
- **原文:** 「処理容量の慢性的超過という共通の基盤から**生じる**異なる**発現形**であることを示唆する」
- **問題類型:** DIRECTIONAL_LANGUAGE
- **問題:** 「示唆する」で hedging しているが、「生じる」「発現形」は因果的メカニズムを前提とした言語。マクロ相関データからメカニズムの同定はできない。
- **修正案:** 「処理容量の慢性的超過という共通の潜在的要因と**関連する可能性がある**ことを示唆する」

### M-04: 知見4 -- 「インターネットの効果」
- **所在:** 日本語版 p.8 (Section 2.1.2)
- **原文:** 「全モデルにおいてインターネット普及率の**係数は正かつ有意**に残った」→ 見出しで「**インターネットの効果**を説明しない」
- **問題類型:** DIRECTIONAL_LANGUAGE
- **問題:** 「効果」(effect) は因果用語。回帰係数は「関連」であり「効果」ではない（Country FE のみで Year FE なしの場合）。
- **修正案:** 「インターネット普及率との**関連**を説明しない」

### M-05: 閾値前後の比較 -- 「鬱病有病率の上昇が約20倍に加速している」
- **所在:** 日本語版 p.8
- **原文:** 「インターネット/教育比が上昇した時期以降、鬱病有病率の上昇が約20 倍に**加速している**」
- **問題類型:** HEDGING_NEEDED
- **問題:** 時系列上の相関を因果的加速と表現。同時期に他の変化（SNS普及、経済変動、診断基準変更等）も起きている。
- **修正案:** 「...以降の期間において、鬱病有病率の変化率が約20倍**大きい**（ただし同時期の他の変化との交絡が排除されていない）」

### M-06: Section 2.2.3 -- 「確認された」
- **所在:** 日本語版 p.12
- **原文:** 「鬱病の測定が経済発展に汚染されていたことが**確認された**」
- **問題類型:** HEDGING_NEEDED
- **問題:** GDPの係数が自殺率ではゼロだが鬱病では巨大であることから「確認」としているが、他の説明（例: 豊かな社会特有のリスク要因）も排除されていない。
- **修正案:** 「...汚染されていた可能性が**強く示唆された**」

### M-07: Section 2.2.4 -- 「保護効果から有害効果への反転」
- **所在:** 日本語版 p.12-13
- **原文:** 「低普及率段階では情報アクセスの恩恵が**支配的**であり、高普及率段階では認知負荷の害が**支配的になる**」
- **問題類型:** DIRECTIONAL_LANGUAGE
- **問題:** 閾値掃引分析で係数の符号が反転したことを「支配的」と因果的に解釈。高普及率サンプルはOECD諸国中心であり、先進国特有の多数の交絡因子が代替説明となりうる。著者自身がこの限界を後述しているが、主文の表現が強すぎる。
- **修正案:** 「低普及率段階では負の係数（保護的関連）が観察され、高普及率段階では正の係数（有害な関連）が観察される。ただし...」

### M-08: Section 2.2.5.2 -- 「精神衛生への効果が真逆になる」
- **所在:** 日本語版 p.15
- **原文:** 「同じ経済成長であっても、それが広告エコシステムの膨張を伴うか否かで、精神衛生への**効果が真逆になる**」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 「効果」は因果用語。1階差分回帰の係数の符号差を「効果が真逆」と表現するのは過剰。
- **修正案:** 「...精神衛生指標との**関連の方向が逆転する**」

### M-09: Section 3.2 -- 「なぜ先進国において特にバランスが病理的になるのか」
- **所在:** 日本語版 p.20 (Section 3.2)
- **原文:** 「なぜ先進国において特にバランスが**病理的になる**のか」
- **問題類型:** DIRECTIONAL_LANGUAGE
- **問題:** 「病理的になる」は因果的過程を前提。相関パターンが先進国で顕著であることは観察されたが、バランスが「病理的になる」メカニズムは推測。
- **修正案:** 「なぜ先進国において特にバランスと精神病理の**関連が顕在化する**のか」

### M-10: Section 3.2.2 -- 「SNSとデジタルメディアは...増幅する」
- **所在:** 日本語版 p.20-21 (Section 3.2.2), 英語版 p.7
- **原文:** 「SNS とデジタルメディアは、2 つのメカニズムでこの非対称性を**増幅する**」/ "Social media and digital media **amplify** this asymmetry through two mechanisms"
- **問題類型:** HEDGING_NEEDED
- **問題:** 「増幅する」は因果的メカニズムの断定。理論セクションとして提案されている文脈は理解できるが、実証データなしに断定的。
- **修正案:** 「SNSとデジタルメディアは、2つのメカニズムでこの非対称性を**増幅すると考えられる**」

### M-11: Section 3.2.3 -- 「教育は圧縮として機能する」
- **所在:** 日本語版 p.21 (Section 3.2.3), 英語版 p.7-8
- **原文:** 「これが教育の回帰係数が負（鬱病低減）である理由を**説明する**」/ "This **explains** why education's regression coefficient is negative"
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** 教育年数の負の回帰係数は多くの代替説明（SES、医療アクセス、生活環境等）がある。「圧縮として機能する」という特定メカニズムが回帰係数を「説明する」とするのは過剰。
- **修正案:** 「これは教育の回帰係数が負である**一つの可能な解釈を提供する**」

### M-12: Section 3.3 -- 「ゲームは感覚運動ループを閉じるが、SNSはそれを開いたままにする」
- **所在:** 日本語版 p.23-24, 英語版 p.9
- **原文:** 「ループ閉合枠組みはより節約的な説明を提供する: ゲームは感覚運動ループを**閉じる**が、SNS はそれを**開いたまま**にする」
- **問題類型:** HEDGING_NEEDED
- **問題:** Boers et al. (2019) の縦断データをループ閉合度で再解釈しているが、Boers et al. のデータからループ閉合度を直接測定していない。「より節約的」という主張は検証なしには評価できない。
- **修正案:** 「ループ閉合枠組みは**別の解釈を提案する**: ゲームは感覚運動ループの閉合度が**比較的高い活動であり**、SNSは**比較的低い活動である**。この区別がBoers et al.の知見を**説明する可能性がある**」

### M-13: Section 6.1 -- 「認知肥満モデルは統合する上位メカニズムを提供する」
- **所在:** 日本語版 p.30, 英語版 p.13
- **原文:** 「認知肥満モデルによる再解釈は、これらを統合する上位メカニズムを**提供する**」/ "The cognitive obesity model **provides** an integrating higher-order mechanism"
- **問題類型:** HEDGING_NEEDED
- **問題:** 他者のデータを自説のレンズで再解釈することは「提供する」ではなく「提案する」。統合的メカニズムが実際に機能しているかは未検証。
- **修正案:** 「認知肥満モデルは、これらを統合する上位メカニズムを**提案する**」

### M-14: Section 6.1 -- 「物質購買の高いバランスが自参照ループを駆動し」
- **所在:** 日本語版 p.30
- **原文:** 「物質購買の高いバランスが自参照ループ（選択反芻）を**駆動し**」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** Carter & Gilovich (2010) のデータを「駆動する」と因果的に再解釈。元研究は物質購買後の反芻を測定したが、「バランス」は測定していない。
- **修正案:** 「物質購買の高いバランスが自参照ループを**促進すると解釈できる**」

### M-15: Section 6.2 -- 「バランス改善がsgPFC-DMN結合を弱め、自参照ループの神経基盤を一時的に切断する」
- **所在:** 日本語版 p.30-31, 英語版 p.13-14
- **原文:** 「このバランス改善がsgPFC-DMN 結合を弱め、自参照ループの神経基盤を一時的に**切断する**と解釈できる」
- **問題類型:** HEDGING_NEEDED
- **問題:** Bratman et al. (2015) のfMRIデータは自然散歩 vs 都市散歩の比較であり、「バランス」変数は測定されていない。「切断する」は強い因果表現。「解釈できる」で hedging しているが、メカニズムの記述部分が断定的すぎる。
- **修正案:** 「このバランス改善がsgPFC-DMN結合の低下と**関連している可能性があり**、自参照ループの神経基盤への**影響として解釈しうる**」

### M-16: Section 8.1 -- 「BAの臨床的成功はバランスメカニズムの間接的な傍証」
- **所在:** 日本語版 p.35, 英語版 p.16
- **原文:** 「BA の臨床的成功...は、本モデルのバランスメカニズムの間接的な**傍証**として位置づけられる」/ "BA's clinical success thus serves as **indirect corroboration** of the balance mechanism"
- **問題類型:** HEDGING_NEEDED
- **問題:** BAの成功は多くのメカニズム（行動強化、社会的接触、達成感等）で説明可能。特定の「バランスメカニズム」の傍証とするのは過剰解釈。
- **修正案:** 「BAの臨床的成功は本モデルと**矛盾しない**が、バランスメカニズムの**固有の証拠とはならない**」

### M-17: Section 3.5.2 -- AI追従性と妄想スパイラルの因果連鎖
- **所在:** 日本語版 p.26-27
- **原文:** 「理想的なベイズ合理的ユーザーさえもAI の追従性によって妄想スパイラルに**陥る**ことを理論的に**証明した**」
- **問題類型:** HEDGING_NEEDED
- **問題:** Chandra et al. のベイズモデルは理論的証明であり、経験的証明ではない。「証明した」はモデル内の論理的帰結としては正しいが、現実世界での因果的証明と混同されうる。
- **修正案:** 「...妄想スパイラルに陥りうることを**理論モデル上で示した**」

### M-18: Section 2.3.2 -- NHANES「運動固有の効果が確認された」
- **所在:** 日本語版 p.17-18
- **原文:** 「SES・健康状態を超えた運動**固有の効果が確認された**」
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** NHANESは横断的観察データ。共変量を統制しても「固有の効果」（causal effect specific to exercise）は同定できない。測定されていない交絡変数（性格特性、社会的ネットワーク等）が残存する。
- **修正案:** 「SES・健康状態を統制した後も運動と鬱の**独立した関連が確認された**」

### M-19: Section 2.3.3 -- 「独立に負の効果を持った」
- **所在:** 日本語版 p.18, 英語版 p.11
- **原文:** 「受動的余暇（TV＋PC）が独立に負の**効果**を持った」/ "passive leisure (TV+PC) independently had **negative effects**"
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** ATUS横断データの回帰係数を「効果」(effects) と表現。横断データでは「効果」は成立しない。
- **修正案:** 「受動的余暇が独立に**負の関連**を示した」/ "passive leisure independently showed **negative associations**"

### M-20: 英語版 Section 6.4 -- 「Direct Causal Evidence for Term 2」
- **所在:** 英語版 p.13 (Section 6.4 見出し)
- **原文:** "RCT Evidence for Experiential Processing Manipulation: **Direct Causal Evidence for Term 2**"
- **問題類型:** CAUSAL_OVERCLAIM
- **問題:** セクション見出しで「Term 2の直接的因果証拠」と主張。運動RCTは「運動→鬱改善」の因果証拠であり、「Term 2（体験的処理）→鬱改善」の因果証拠ではない。運動をTerm 2と等値する操作は著者の理論的解釈であり、検証されていない。
- **修正案:** "RCT Evidence for Exercise Interventions: **Indirect Support for the Role of Term 2**"

## Minor (軽微)

### m-01: 導入 -- 「認知の肥満がカロリー摂取と消費の加法的バランス不均衡から生じるように」
- **所在:** 日本語版 p.4
- **原文:** 「身体の肥満がカロリー摂取と消費の加法的バランス不均衡（EB = Intake − Expenditure）から**生じる**ように」
- **問題類型:** CAUSAL_APPROPRIATE
- **備考:** 肥満とエネルギーバランスの因果関係は確立されており、ここでの「生じる」はアナロジーの基盤として適切。ただし、アナロジーからターゲット（認知の肥満）への転用は検証が必要。

### m-02: 「予測因子」の使用
- **所在:** 日本語版 p.6 (知見2), 他多数
- **原文:** 「インターネット/教育比が鬱病の**予測因子**として機能する」
- **問題類型:** DIRECTIONAL_LANGUAGE
- **備考:** 「予測因子」(predictor) は回帰分析の標準用語であり技術的には適切。ただし一般読者には因果的な含意を持ちうる。文脈から許容範囲。

### m-03: Section 3.1 -- 「このアナロジーは単なる比喩ではない」
- **所在:** 日本語版 p.20, 英語版 p.7
- **原文:** 「このアナロジーは単なる比喩ではない」/ "This analogy is not merely metaphorical."
- **問題類型:** HEDGING_NEEDED
- **問題:** アナロジーが「比喩以上」であることの根拠として「検証可能な予測を生成する」を挙げているが、予測を生成することと予測が正しいことは別。
- **修正案:** 「このアナロジーは単なる比喩にとどまらず、**検証可能な予測を生成する点で理論的構造を持つ**（ただしその予測の妥当性は今後の検証に依存する）」

### m-04: Section 3.2.4 -- 「鬱病 = 処理コストが処理容量を慢性的に超過した状態」
- **所在:** 日本語版 p.21, 英語版 p.8
- **原文:** 「鬱病 = 処理コストが処理容量を慢性的に超過した状態」/ "Depression = chronic exceedance of processing cost over processing capacity."
- **問題類型:** HEDGING_NEEDED
- **問題:** 等号で定義しているが、鬱病の定義としては過度に還元的。生物学的要因（遺伝、神経化学）を排除する印象を与える。
- **修正案:** 「本モデルでは、鬱病を処理コストが処理容量を慢性的に超過した状態として**モデル化する**（鬱病の多因子的性質を排除するものではない）」

### m-05: Section 3.2.6 -- 「疫学的変数としては極めて高い説明力」
- **所在:** 日本語版 p.22-23, 英語版 p.8
- **原文:** 「within R² = 0.252。...疫学的変数としては**極めて高い説明力**」/ "exceptionally high for an epidemiological variable"
- **問題類型:** HEDGING_NEEDED
- **問題:** within R² = 0.252 を「極めて高い」と評価しているが、これは Country FE のみ（Year FE なし）のモデル。TWFE では第一近似が崩壊しているため、この数値の解釈は慎重であるべき。
- **修正案:** 「within R² = 0.252（ただしこれはCountry FEのみのモデルであり、Year FE追加で崩壊した第一近似のスケールでの値である点に注意）」

### m-06: Section 2.2.5 -- 「情報技術の精神衛生への影響」
- **所在:** 日本語版 p.14
- **原文:** 「情報技術の精神衛生への**影響**が「量」ではなく「質」...に依存することを示唆する」

## 構造的問題 (Structural Issues)

### S-01: Ecological Fallacy の取り扱い
- **問題:** 著者はecological fallacyへの注意を繰り返し明記しており（Abstract、Section 2.1.3、Section 2.4、Section 8.4）、この点は誠実である。しかし、限界の記述と本文中の因果的表現の間にトーンの乖離がある。限界セクションでは「個人レベルのメカニズムに直接帰属できない」と述べながら、本文の分析結果の解釈では「駆動する」「生じる」「支配する」等の因果言語を使用している。読者は限界セクションよりも本文中の解釈を記憶する傾向があるため、因果的印象が残りやすい。
- **推奨:** 本文中の結果解釈において、限界セクションと同等の慎重さを一貫して維持する。

### S-02: 他者のRCTデータの再解釈
- **問題:** Section 6全体が既存研究の「バランスモデルによる再解釈」であるが、再解釈と検証の区別が不十分な箇所がある。特にSection 6.4（運動RCT）では、運動の因果効果を「Term 2の因果証拠」として転用している。運動RCTは「運動→鬱改善」を示すが、「体験的処理の増大→鬱改善」は著者の理論的等値であり、この等値自体が検証対象。
- **推奨:** Section 6の冒頭にある「以下は既存データの再解釈であり、バランスメカニズムの直接的因果検証ではない」という注記を各小節の結論部でも繰り返す。

### S-03: 横断データ（NHANES, ATUS）の位置づけ
- **問題:** NHANESとATUSの分析は "Individual-level validation" (英語版) / 「個人レベルの検証」(日本語版) として位置づけられているが、いずれも横断的観察データであり "validation" は過剰。横断データは関連の存在を示すが、メカニズムの妥当性検証 (validation) には不十分。
- **推奨:** "Individual-level cross-sectional analysis" / 「個人レベルの横断的分析」に変更し、"validation" / 「検証」の語を避ける。

### S-04: 「整合する」(consistent with) の累積的問題
- **問題:** 論文全体で「整合する」「予測と一致する」等の表現が20回以上使用されている。個々は適切なhedgingだが、累積的に読むと「あらゆるデータが本理論を支持している」という印象を形成する。しかし、ほとんどの結果は代替理論（displacement hypothesis、social comparison theory、stress-diathesis model等）とも整合する。
- **推奨:** 主要な知見ごとに、バランスモデル以外の代替説明を明示的に記述し、バランスモデルがそれらに対してどのような追加的説明力を持つか（または持たないか）を議論する。

### 全体評価

本論文の因果言語に関する全体的評価は以下の通り。

**肯定的側面:**
1. 著者はecological fallacyへの意識を繰り返し表明しており、限界セクション（Section 2.1.3, 8.4等）は誠実かつ詳細である。
2. 英語版AbstractのSection 8.4 ("Honest Assessment of Current Evidence") は、現時点の証拠の限界を率直に認めた模範的なセクションである。
3. 反証条件（Section 7）の明示は、科学的誠実さの表れとして高く評価できる。
4. TWFE による第一近似の崩壊を「失敗ではなく教訓」として報告する姿勢は、HARKingを回避する好例。
5. 多くの箇所で「示唆する」「整合する」等の適切なhedgingが使用されている。

**問題点:**
1. **Abstract と限界セクションのトーン乖離**: Abstract では「規定因」「確認された」等の強い因果言語を使用しながら、本文の限界セクションでは生態学的誤謬や因果未確定を認めている。読者はAbstractの印象で論文を評価するため、この乖離は実質的な問題。
2. **横断データ（NHANES, ATUS）の因果的解釈**: 「効果」「確認された」「固有の効果」等、横断データの限界を超えた表現が散見される。
3. **他者のRCTの理論的転用**: 運動RCTの因果証拠を「Term 2の因果証拠」に転用する論理は、著者の理論的等値が検証済みであることを前提としており、循環的。
4. **マクロパネルデータの因果言語**: 「効果」「駆動する」「支配する」等、パネルデータの相関・回帰結果に対する因果的表現が多い。
5. **「整合する」の過剰使用**: 個々は適切でも累積的にconfirmation biasの印象を与える。

**優先度の高い修正:**
1. Abstract全体の因果言語を限界セクションのトーンに合わせる（特にC-01, C-02, C-05）
2. NHANES/ATUSの結果記述から「効果」「確認」を「関連」「観察」に置換する（C-05, C-11, M-18, M-19）
3. Section 6.4の見出しと主文から「直接的因果証拠」を「間接的支持」に変更する（C-10, M-20）
4. ラグ分析の「因果」を「予測的先行関係」に統一する（C-08）

**著者への注記:**
本論文は探索的研究として明示されており、仮説生成を主目的としている点は適切に位置づけられている。因果言語の修正は、論文の主張を弱めるものではなく、むしろ知的誠実さを強化し、査読者からの信頼を高めるものである。特にAbstractの表現を慎重にすることで、本文の限界セクションとの一貫性が保たれ、論文全体のcredibilityが向上する。
